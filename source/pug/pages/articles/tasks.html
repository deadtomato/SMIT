• 1. Подготовка к работе

    • 1. Сделайте форк проекта
    • 2. Затем, клонируйте свой репозиторий (форк) проекта к себе на компьютер.
    • 3. Создайте ветку task1 в склонированном проекте.
    • 4. В этой ветке инициализируйте npm-пакет.
    • 5. Установите gulp и плагины (gulp-sass, gulp-plumber, gulp-postcss, gulp-sourcemaps, autoprefixer,browser-sync)
    • 6. Создайте и сконфигурируйти gulpfile.js
    • 7. Напишите скрипты в packages.json
    • 8. Создайте структуру проекта в папке source. В пустых папках img, fonts и т.д. создайте файл README (БЕЗ РАСШИРЕНИЯ!), в котором укажите, что должно быть в этой папке. К примеру, в папке img должен лежать файл README с текстом: "Папка для изображений"
    • 9. Создайте в source файл index.html. Создайте базовую разметку. В body создайте заголовок первого уровня с текстом Hello world!
    • 10. Выполните тестовый запуск локального сервера, для проверки результата.
    • 11. Создайте пулреквест из ветки task1 вашего форка в мастер-репозиторий.
    • Если требуются дополнительные изменения, то их нужно вносить в ветке из которой создан пулреквест. Новый пулреквест создавать не нужно. Все изменения автоматически появятся в уже созданном пулреквесте.
