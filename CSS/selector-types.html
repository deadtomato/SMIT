Виды селекторов

Селектор Значение Пример
Универсальный селектор Применяется ко всем элементам в документе 
                
* {
  margin: 0;
  padding: 0;
}
Селектор тегов Применяется к указанным тегам 
                
h1, h2, h3 {

}
Селектор классов Применяется тегам с классом class="className" 
                
.className {
                    
}
                
              
                
p.className {

}
Селектор идентификаторов Применяется к тегу c идентификатором id="idName" 
                
#idName {

}
Селектор атрибутов Применяется к тегам c указанным атрибутом 
                
input[href="someSite"] {
  
}
Селектор потомков (контекстный селектор) Применяется к элементу, который является потомком другого элемента 
                
p a {
  
}

.class .class {

}
Селектор дочерних элементов Применяется к элементу, который является прямым потомком другого элемента 
                
li > a {
  
}
Селектор соседних элементов Применяется к элементу, который следует за элемента 
                
h1 + p {
  
}
Селектор последующих элементов Применятся к элементу, только если перед ним расположен данный элемент 
                                    
h1 ~ p {
                      
}
Селектор псевдоклассов Применятся к элементу, содержащиму указанный псевдокласс 
                                    
a:hover {
                      
}
Селектор псевдоэлементов Применятся к элементу, содержащиму указанный псевдоэлемент 
                                    
p::first-letter {
                      
}
